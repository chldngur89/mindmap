- README가 없는 프로젝트에서 더 많은 소스 파일을 읽도록 확장

- 분석 결과에 근거 파일 경로를 더 자주 포함

- 생성된 README에 실행 예시와 API 설명 추가

- 프로젝트 폴더에 맵 JSON 직접 저장

- 자주 보는 폴더 즐겨찾기

### 4. AI 품질 개선

- `exaone3.5` 사용 시 프롬프트 미세 조정

- 어시스턴트 결과에 더 강한 fallback / 재시도 전략 추가
